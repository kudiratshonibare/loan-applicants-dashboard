# Loan Applicants Dashboard

## Overview
This project presents an interactive Power BI dashboard analyzing loan applicant characteristics and loan request patterns.

## Objective
To provide insights into applicant demographics, income levels, education, property areas, and loan amounts to support business decision-making.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Urban applicants requested the highest total loan amount.
- Most applicants were graduates.
- Male applicants accounted for the majority of applicants.
- Applicants without dependents requested the highest loan amounts.
